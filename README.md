Who Wants to Be a Millionaire Telegram Bot

Telegram-бот, реализующий популярную интеллектуальную игру «Кто хочет стать миллионером». Пользователь отвечает на вопросы разной сложности, зарабатывает виртуальные деньги и может использовать игровые подсказки.

Возможности

* Регистрация игроков
* Система вопросов и ответов
* Несколько уровней сложности
* Подсчет выигрыша
* Проверка правильности ответов
* Хранение статистики игроков
* Удобное взаимодействие через Telegram

Технологии

Backend

* Python
* Aiogram
* SQLite
* SQLAlchemy

Инструменты

* Git
* GitHub
* Telegram Bot API

Запуск проекта

1. Клонирование репозитория

git clone https://github.com/your-username/millionaire-bot.git
cd millionaire-bot

2. Создание виртуального окружения

python -m venv .venv

Windows:

.venv\Scripts\activate

Linux/macOS:

source .venv/bin/activate

3. Установка зависимостей

pip install -r requirements.txt

4. Настройка переменных окружения

Создайте файл .env:

BOT_TOKEN=your_telegram_bot_token

5. Запуск бота

python run.py

## 
1. Пользователь запускает бота командой /start.
2. Бот предлагает начать игру.
3. Игрок получает вопрос и варианты ответов.
4. При правильном ответе переходит на следующий уровень.
5. При ошибке игра завершается.
6. Результаты сохраняются в базе данных.

Особенности проекта

* Асинхронная архитектура на Aiogram.
* Работа с базой данных SQLite.
* Хранение статистики игроков.
* Реализация игровых механик оригинальной телевизионной игры.
* Чистая модульная структура проекта.
